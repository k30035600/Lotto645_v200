모바일 뷰포트 근사 미리보기 (iframe)

실제 Note10 화면과 완전히 같지 않습니다. 이 페이지는 PC 브라우저 안에 CSS 픽셀 폭·높이만 맞춘 iframe입니다. 아래는 대표적인 차이입니다.
  • 브라우저 엔진·삼성 인터넷·카카오 인앱 등 렌더링·폰트·줌이 다릅니다.
  • 간편 모드·디스플레이 크기·최소 너비(DP) 설정은 여기서 재현되지 않습니다.
  • 주소창·제스처·노치·DPR(밀도)는 iframe과 다릅니다.
진짜 기기와 맞추려면 폰에서 Chrome 원격 디버깅 등으로 window.innerWidth, window.innerHeight 를 재서 아래 숫자에 넣는 것이 가장 가깝습니다.

서버: python server.py 후 이 문서는 http://127.0.0.1:8000/dev_note10_viewport.html 로 엽니다. KakaoTalk 캡처 1080×2156 과 같은 종횡비를 쓰려면 세로 = 가로 × (2156÷1080) ≈ 가로 × 1.996 을 쓰면 됩니다.
