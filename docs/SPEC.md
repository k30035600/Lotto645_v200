# 로또 번호 생성기 기능 명세 (V2)

## 1. 홀짝 (Odd/Even)

- **홀수와 짝수의 개수 비율**
- **홀짝없음 (none)**: 비율 제한 없음
- **홀 1~5, 짝 5~1**까지 조합으로 매핑

---

## 2. 연속 (Sequence)

- **연속된 숫자 쌍의 개수**: 1회, 2회, 3회로 매핑
- **연속없음 (none)**: 연속된 쌍 없음
- **1회**: (1,2) 또는 (10,11)
- **2회**: (1,2,3) 또는 (10,11 and 20,21)
- **3회**: (1,2,3,4) 또는 (10,11 and 20,21 and 30,31)

---

## 3. 핫콜 (Hot/Cold)

- **과거 당첨 횟수 통계** 기준: 상위 50% = **핫(Hot)**, 하위 50% = **콜(Cold)**
- **핫콜없음 (none)**: 비율 제한 없음
- **핫 1~10, 콜 10~1**까지 조합으로 매핑

---

## 4. 모드 선택 → 구간선택

- **모드선택을 구간선택으로 변경**: 게임공 6개의 **합 구간**을 선택
- **구간선택 UI**: "시작~종료" 출력, 각 **3자리 숫자** 입력
- **시작값**
  - 최소: (1+2+3+4+5+6) = **21** 이상
  - 기본값: 회차별 당첨번호 합계 중 **제일 적은 값**
- **종료값**
  - 최대: (40+41+42+43+44+45) = **255** 이하
  - 기본값: 회차별 당첨번호 합계 중 **제일 큰 값**

---

## 5. 게임 세트 컨테이너 (1게임 ~ 5게임, 총 5개)

### 5.1 각 게임의 모드 버튼

| 모드 | 설명 |
|------|------|
| **자동** | 홀짝/연속/핫콜/구간선택 조건에 따라 6개 번호 자동 생성. 6개 번호의 합이 구간에 포함되면 유지, 벗어나면 **재생성** |
| **반자동** | 자동으로 선택된 6개 번호를 기본으로 사용. 선택 번호 취소·변경 가능, 번호선택 그리드에서 교체. 선택된 번호 **오름차순 정렬**. 반자동일 경우 **시작/종료(구간) 확인 skip** |
| **수동** | 6개 번호 초기화 후, 번호선택 그리드에서 6개 직접 선택. 선택된 번호 **오름차순 정렬** |

### 5.2 체크박스

- **6개 공이 모두 선택되면** 체크박스 **활성화**
- **체크 시**: 번호 저장 준비 완료, 해당 게임의 **모드 버튼 비활성화**
- **체크 해제 시**: 해당 게임의 **모드 버튼 활성화**

---

## 6. 저장 박스

- **1게임~5게임의 체크박스가 모두 체크**되면 저장 영역 **활성화**

### 6.1 저장 회차·세트 입력

- **저장회차**: **최종회차 + 1**을 회차에 출력
- **세트번호**: **1**을 세트에 출력 (기본)
- 회차와 세트번호가 입력되면 **"저장" 버튼 활성화**

### 6.2 저장 버튼

- **저장** 선택 시:
  - 선택된 각 게임의 **회차, 세트, 게임, (홀짝/연속/핫콜/게임모드), 선택공 6개**를 **결과 박스(#resultBox)**에 출력
  - **Lotto023.xlsx**에 저장

---

## 7. 결과 박스 (#resultBox)

- **Lotto023.xlsx**을 읽어 출력
- **표시 항목**: 회차, 세트, 게임, (홀짝/연속/핫콜/게임모드), 선택공 6개
- **보너스공**: 해당 회차 추첨이 끝난 경우에만, 해당 회차의 보너스 번호 출력
